# 使用在线工具转换文件

由于无法直接在本地安装 python-pptx 库，我建议使用在线工具来转换文件。以下是一些常用的在线工具：

## 1. SmallPDF

- 网站：https://smallpdf.com/
- 支持的格式：PDF、Word、Excel、PowerPoint 等
- 优点：界面简单，支持多种格式
- 缺点：免费用户有文件大小和数量限制

## 2. Zamzar

- 网站：https://www.zamzar.com/
- 支持的格式：超过 1200 种格式
- 优点：支持多种格式，转换速度快
- 缺点：免费用户有文件大小和数量限制

## 3. Convertio

- 网站：https://convertio.co/
- 支持的格式：超过 300 种格式
- 优点：界面简单，支持多种格式
- 缺点：免费用户有文件大小和数量限制

## 4. Online-Convert

- 网站：https://www.online-convert.com/
- 支持的格式：超过 200 种格式
- 优点：支持多种格式，转换速度快
- 缺点：免费用户有文件大小和数量限制

## 使用步骤

1. 访问上述网站之一
2. 选择要转换的文件（/home/lmguo/桌面/1_2_简历-郭黎明.docx）
3. 选择要转换的格式（PowerPoint/PPTX）
4. 点击转换按钮
5. 下载转换后的文件
6. 将文件保存到 /home/lmguo/桌面 目录下

## 注意事项

- 确保您的文件不包含敏感信息，因为在线工具可能会保留您的文件
- 免费用户可能会有文件大小和数量限制
- 如果您需要转换大量文件，建议使用付费工具或本地安装的工具
